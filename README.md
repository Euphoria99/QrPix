# QrPix

<p align="center">
    QrPix is a web app to generate QR codes with images.
</p>

<p align="center">
    <img src="images/qrpixlogo.png">
</p>

### Usage

1. Open [site](https://euphoria99.github.io/QrPix/).
2. Upload an image.
3. Enter the URL.
4. Adjust parameters.
5. Download `PNG`.
